
# Guess the Word Game

This is a word-guessing game where the player tries to guess the correct word based on a set of scrambled letters. The game tracks how many rounds it takes for the player to guess the correct word. After guessing correctly, the game displays the word and offers a restart button to play again.

## Installation

No installation is required. The project runs directly in your web browser.

### Prerequisites

- A modern web browser (Google Chrome is recommended).

## Usage

1. Open the `index.html` file in your browser.
2. Click the **Start** button to begin the game.
3. You will see scrambled letters that form a word.
4. Enter your guess in the input field provided.
5. After each guess:
   - If your guess matches the correct word, the game will display how many rounds it took you to guess correctly and show the correct word.
   - If your guess is incorrect, continue guessing until you get it right.
6. Once you guess correctly, the game offers a **Restart** button to play again.

## Features

- **Word Scramble**: Provides scrambled letters that form a word for the player to guess.
- **Guess Tracking**: Tracks the number of rounds it takes for the player to guess the correct word.
- **Display Correct Answer**: Reveals the correct word after it has been guessed.
- **Start and Restart Buttons**: Start the game with a **Start** button and reset the game after completion with a **Restart** button.
